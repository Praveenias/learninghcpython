# Raw Bom - Atom BOM Conversion
* Raw Bom - Atom BOM Conversion is a Python project to automatically convert the RAW BOM file into ATOM BOM template. 
* The project also generates customer part number automatically. 

# Execution
1. Run the exe file
2. Provide the input and output directory path 
3. Click Submit Button
4. Code execution concludes with Task Completed Successfully Message
5. Converted ATOM BOM files will be available in the chosen output folder.
