
# ATOM LAB

## Directory Structure

  * src -- Source files  
  * data -- Any data file require for the programs (eg) Excel, CSV files  
  * examples -- Maintain any example files taken from other sources  
  * doc -- If any auto documentation generated  
  * logs -- If our source file generate any logs  
  * tmp -- Temporary files, these will not be part of git repository  

This directory structure will evolve as project progress
